# Weaver-Yuwono M3 Design System

A modular, high-end editorial design system based on Material 3. Designed for the Weaver-Yuwono Family Office.

## Aesthetic: "Soft Modernism"
- **Typography:** Playfair Display (Serif) for headings, Manrope (Sans) for body and UI.
- **Palette:** Hunter Green, Alabaster, Warm Clay.
- **Components:** Custom Web Components (`wy-*`) built with Lit and Material Web.

## Layouts
- **Landing Page:** Professional monogram and portfolio overview.
- **Profile Page:** Modular asset and bio modules.
- **Project Detail:** Magazine-style hero and thesis.
- **Portfolio Dashboard:** High-end financial analytics and metrics.

## Development
```bash
npm install
npm run dev
```
